วาง Script URL ที่ได้จาก Google Apps Script เพื่อให้ข้อมูลซิงก์ข้ามเครื่องได้

วิธี deploy Apps Script (ทำครั้งเดียว ~5 นาที):
1. เปิด script.google.com → New Project
2. Copy โค้ดจากไฟล์ jppp-apps-script.js วางทับทั้งหมด
3. บันทึก → Deploy → New deployment → Web app
4. Execute as: Me · Who has access: Anyone
5. Copy URL มาวางด้านล่าง
